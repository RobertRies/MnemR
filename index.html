MnemR

Chunk Learning


Description:

Edit & file tiny bits of media.
Apply study techniques.

Core Feats:

  • draw material using notepad, mic, cam & the web
  • combine flashcard- & table-learning techniques

Side Feats:

  • infinite cells
  • multiple views
  • multiple formats
  • multiple navigations
  • qick & easy IME change




Download:




Build:

 $ git clone git://github.com/RobertRies/MnemR
 $ android update project --path MnemR
 $ cd MnemR && ant install
